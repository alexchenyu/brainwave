  💡 降低成本的策略

  1. 使用缓存
    - 对于重复的系统提示词，使用缓存可节省 98% 的输入成本
    - 从 $0.048/分钟 降至 $0.0006/分钟
  2. 混合方案
    - 语音转文本：用 Whisper API ($0.006/分钟)
    - 文本处理：用 GPT-4o ($0.005/1K tokens)
    - 总成本约 $0.01/分钟（节省 93%）
  3. VAD（语音活动检测）
    - 只在用户说话时发送音频
    - 可节省 50% 的输入成本
  4. 本地语音识别
    - 用浏览器的 Web Speech API
    - 完全免费，但质量略低
